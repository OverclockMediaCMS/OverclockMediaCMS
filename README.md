# OverClockMedia CMS Vite-React-PWA app
Mobile Apps CMS project front-end app

## Setup
1. Clone Repo
2. run "npm install" to install all packages
3. run "npm run dev" to launch

## Layout
* This is just a scaffold, lacking any real styling or functionality! 
* Entrypoint is main.tsx but that just imports and runs app as defined in App.tsx.
* assets, like the images used for the buttons stored in the /assets folder.
* I have setup navigation using routers, these are all defined in App.tsx and navigated between using the navbar element found in utilities/Navbar.tsx. This is just pictures with an onclick event.
* The screens have all be declared seperately in the /screens folder. I think this would be a decent way to structure the project, we should definitely have a discussion and brainstorm as well though.
